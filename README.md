◯ Car Racing Game

∙ Game Rules: Use the arrow keys to race.
∙ Controls
 - Movement: Arrow keys or W, A, S, D
 - Booster: Z
 - Map: M
 - Drift: Ctrl
 - Brake: Shift
 - Restart: Spacebar
